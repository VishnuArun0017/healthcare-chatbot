## Overview

Pregnancy tests detect **hormone hCG** in urine or blood.  
Home tests are accurate when used **after a missed period**.

## Common symptoms suggesting pregnancy

- Missed period  
- Nausea, tiredness  
- Breast tenderness  

## Self-care

- Follow test instructions carefully  
- Use first-morning urine for best accuracy  

## Red flags

- Positive test with pain or bleeding → seek care urgently
